Comandos básicos de ubuntu.

ls: Lista arquivos e diretórios no diretório atual.

cd: Navega entre diretórios. Por exemplo, para entrar no diretório "Documentos".

pwd: Mostra o caminho completo do diretório atual.

mkdir: Cria um novo diretório. Por exemplo, para criar um diretório chamado "projetos".

sudo: Executa comandos com privilégios de superusuário (root). É útil para tarefas administrativas. Por exemplo, para atualizar o sistema.

cp: Copia arquivos ou diretórios. Por exemplo, para copiar um arquivo.

mv: Move ou renomeia arquivos ou diretórios. Para mover um arquivo.

rm: Remove arquivos. Cuidado ao usar esse comando, pois ele apaga permanentemente os arquivos.

apt-get: Gerencia pacotes, usado para instalar, atualizar e remover software. Por exemplo, para instalar um programa.

touch: Cria um novo arquivo vazio. Por exemplo, para criar um arquivo chamado "novo_arquivo.txt".
